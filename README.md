# Card Design Generator
## env
parcel + pug + scss + typescript
 ## install dependencies
```bash
yarn
# or
npm install
```
 ## run dev server
```bash
yarn dev
# or
npm run dev
```
 ## build
```bash
yarn build
# or 
npm run build
```
